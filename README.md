# Sidequest Supply Website

A simple website for selling 3D printed keychains and other items.

## Getting Started

1. Open `index.html` in your web browser to view the homepage.

## Structure

- `index.html`: The main homepage
- `styles.css`: Basic styling
- `fonts/`: Directory for custom fonts (if any)

## Next Steps

- Add product images and descriptions
- Implement shopping cart functionality
- Add more pages (about, contact, etc.)

## Troubleshooting

- If the page doesn't load, ensure you're opening `index.html` from the correct directory.
- For styling issues, check `styles.css` for syntax errors.
